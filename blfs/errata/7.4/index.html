Errata for BLFS Version 7.4

Miscellaneous Errata

  • On the emacs page, the reference url to the libungif package has changed. It should be http://directory.fsf.org/wiki/Libungif.

  • The target of the url to the BLFS bootscripts is not correct. It should be http://www.linuxfromscratch.org/blfs/downloads/7.4/blfs-bootscripts-20130908.tar.bz2. However, the newer version of the boot scripts that is specified in the development version should work without problems.

  • When the 7.4 version of BLFS was generated, the URLs for the patches were inadvertantly left pointing to the SVN patches. As the SVN version of the book is updated, some of these URLs are no longer current. All patches for version 7.4 of BLFS are located at http://www.linuxfromscratch.org/blfs/downloads/7.4/patches/

  • On the subversion page, the instructions for Python bindings should be corrected. Replace:
        make swig-py
    with:
        make swig-py \
             swig_pydir=/usr/lib/python2.7/site-packages/libsvn \
             swig_pydir_extra=/usr/lib/python2.7/site-packages/svn
    and same for make install-swig-py.
