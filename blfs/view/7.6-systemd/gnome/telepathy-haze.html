telepathy-haze-0.8.0

Introduction to Telepathy Haze

Telepathy Haze is a connection manager for the Telepathy framework using libpurple (Pidgin) protocols.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://telepathy.freedesktop.org/releases/telepathy-haze/telepathy-haze-0.8.0.tar.gz

  • Download MD5 sum: b9ee3638833fb50db6276d1b771820b0

  • Download size: 568 KB

  • Estimated disk space required: 5.3 MB

  • Estimated build time: 0.1 SBU

Telepathy Haze Dependencies

Required

Pidgin-2.10.9 and telepathy-glib-0.24.1

Optional

Twisted

Note

The protocols supported by this package depend on the dependencies that were present when building Pidgin-2.10.9.

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/telepathy-haze

Installation of Telepathy Haze

Install Telepathy Haze by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a testsuite.

Now, as the root user: 

make install
