gnome-documents-3.12.1

Introduction to GNOME Documents

GNOME Documents is a document manager which supports local documents (using the Tracker indexing engine), Google Documents and SkyDrive.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gnome-documents/3.12/gnome-documents-3.12.1.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gnome-documents/3.12/gnome-documents-3.12.1.tar.xz

  • Download MD5 sum: ac15852ebd353d91d4f82e41f66bc4d9

  • Download size: 700 KB

  • Estimated disk space required: 18 MB

  • Estimated build time: 0.3 SBU

GNOME Documents Dependencies

Required

Evince-3.12.2, Gjs-1.40.1, gnome-desktop-3.12.2, libgdata-0.15.2, libzapojit-0.0.3, and Tracker-1.0.4

Runtime Dependencies

gnome-online-miners-3.12.0

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gnome-documents

Installation of GNOME Documents

Install GNOME Documents by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: gnome-documents
Installed Libraries: None
Installed Directories: /usr/lib/gnome-documents /usr/share/gnome-documents, and /usr/share/help/*/gnome-documents

Short Descriptions

gnome-documents

is a document manager application for GNOME.
