gnome-photos-3.12.1

Introduction to GNOME Photos

The GNOME Photos package contains an application used to access, organize and share your photos on GNOME.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gnome-photos/3.12/gnome-photos-3.12.1.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gnome-photos/3.12/gnome-photos-3.12.1.tar.xz

  • Download MD5 sum: f1480bec5b07d86ed45a9d667677db1e

  • Download size: 692 KB

  • Estimated disk space required: 20 MB

  • Estimated build time: 0.3 SBU

GNOME Photos Dependencies

Required

desktop-file-utils-0.22, Exempi-2.2.2, gegl-0.2.0, gfbgraph-0.2.2, gnome-desktop-3.12.2, Grilo-0.2.11, Little CMS-2.6, libexif-0.6.21, librsvg-2.40.3, and Tracker-1.0.4

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gnome-photos

Installation of GNOME Photos

Install GNOME Photos by running the following commands: 

./configure --prefix=/usr --docdir=/usr/share/doc/gnome-photos-3.12.1 &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
