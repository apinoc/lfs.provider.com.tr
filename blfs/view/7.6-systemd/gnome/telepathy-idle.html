telepathy-idle-0.2.0

Introduction to Telepathy Idle

Telepathy Idle is an IRC connection manager for the Telepathy framework.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://telepathy.freedesktop.org/releases/telepathy-idle/telepathy-idle-0.2.0.tar.gz

  • Download MD5 sum: 92a2de5198284cbd3c430b0d1a971a86

  • Download size: 556 KB

  • Estimated disk space required: 6.9 MB

  • Estimated build time: 0.2 SBU

Telepathy Idle Dependencies

Required

telepathy-glib-0.24.1

Optional

Twisted

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/telepathy-idle

Installation of Telepathy Idle

Install Telepathy Idle by running the following commands: 

./configure --prefix=/usr &&
make

To test the results, issue: make check.

Now, as the root user: 

make install
