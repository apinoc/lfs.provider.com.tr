gnome-clocks-3.12.0

Introduction to GNOME Clocks

The GNOME Clocks package contains a simple clock application for GNOME.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gnome-clocks/3.12/gnome-clocks-3.12.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gnome-clocks/3.12/gnome-clocks-3.12.0.tar.xz

  • Download MD5 sum: 7d6354501773c034e49c912169abfd6c

  • Download size: 824 KB

  • Estimated disk space required: 11 MB

  • Estimated build time: 0.2 SBU

GNOME Clocks Dependencies

Required

gnome-desktop-3.12.2, GeoClue-0.12.0, geocode-glib-3.12.2, libcanberra-0.30, libgweather-3.12.2, and Vala-0.24.0

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gnome-clocks

Installation of GNOME Clocks

Install GNOME Clocks by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: gnome-clocks
Installed Libraries: None
Installed Directories: /usr/share/gnome-clocks and /usr/share/help/*/gnome-clocks
