Sushi-3.12.0

Introduction to Sushi

The Sushi package contains a quick file previewer for Nautilus.

This package is known to build and work properly using an LFS-7.6 systemd platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/sushi/3.12/sushi-3.12.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/sushi/3.12/sushi-3.12.0.tar.xz

  • Download MD5 sum: 97bb127c9a66dfc38d72ba6c9f04d33f

  • Download size: 276 KB

  • Estimated disk space required: 5.6 MB

  • Estimated build time: 0.1 SBU

Sushi Dependencies

Required

clutter-gst-2.0.12, clutter-gtk-1.5.4, Evince-3.12.2, Gjs-1.40.1, gst-plugins-base-1.4.1, gtksourceview-3.12.3, libmusicbrainz-5.0.1, and WebKitGTK+-2.4.5

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/sushi

Installation of Sushi

Install Sushi by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a testsuite.

Now, as the root user: 

make install

Contents

Installed Programs: sushi
Installed Libraries: None
Installed Directories: /usr/lib/sushi and /usr/share/sushi

Short Descriptions

sushi

is a graphical file viewer.
