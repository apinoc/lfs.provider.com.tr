kdegames-3.3.2

Introduction to kdegames

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.3.2/src/kdegames-3.3.2.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.3.2/src/kdegames-3.3.2.tar.bz2

  • Download MD5 sum: 41791396e595b9fc8a84e08ae63b552d

  • Download size: 9.3 MB

  • Estimated disk space required: 99.4 MB (additional 15.6 MB for API docs)

  • Estimated build time: 7.14 SBU (additional 0.16 SBU for API docs)

kdegames dependencies

Required

kdebase-3.3.2

Recommended

libjpeg-6b and libxml2-2.6.17

Optional

GraphViz and Doxygen-1.4.1

Installation of kdegames

Install kdegames with: 

./configure --prefix=$KDE_PREFIX --disable-debug \
    --disable-dependency-tracking &&
make

Note

If you wish to create the API documentation and you have Doxygen and GraphViz installed, make apidox must be done before make install.

Now, as the root user: 

make install
