kdeartwork-3.3.2

Introduction to kdeartwork

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.3.2/src/kdeartwork-3.3.2.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.3.2/src/kdeartwork-3.3.2.tar.bz2

  • Download MD5 sum: 9a712da253bacb87e0d4fd28cec183a1

  • Download size: 17.9 MB

  • Estimated disk space required: 105 MB

  • Estimated build time: 1.96 SBU

kdeartwork dependencies

Required

kdebase-3.3.2

Recommended

libjpeg-6b, libxml2-2.6.17, and libart_lgpl-2.3.16

Optional

XScreenSaver-4.19

Installation of kdeartwork

Install kdeartwork with: 

./configure --prefix=$KDE_PREFIX --disable-debug \
    --disable-dependency-tracking &&
make

Now, as the root user: 

make install
