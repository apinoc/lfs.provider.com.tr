libdv-0.103

Introduction to libdv

libdv (Quasar DV) is a software CODEC for DV video, the encoding format used by most digital camcorders.

Package information

  • Download (HTTP): http://prdownloads.sourceforge.net/libdv/libdv-0.103.tar.gz

  • Download (FTP):

  • Download MD5 sum: b5c7d0db672a76528e100828dd866a79

  • Download size: 520 KB

  • Estimated disk space required: 9.4 MB

  • Estimated build time: 0.21 SBU

libdv dependencies

Optional

popt-1.7-5, pkgconfig-0.15.0, SDL-1.2.8, GTK+-1.2.10 and X (XFree86-4.4.0 or X.org-6.8.2)

Installation of libdv

Install libdv by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

Note

The configure check for GTK+ is broken. If GTK+ is not installed, also pass --disable-gtk to the configure script.

Contents

The libdv package contains dubdv, dvconnect, encodedv, playdv and libdv.

Description

dubdv

dubdv inserts audio into a digital video stream.

dvconnect

dvconnect is a small utility to send or capture raw data from and to the camcorder.

encodedv

encodedv encodes a series of images to a digital video stream.

playdv

playdv displays digital video streams on the screen.

libdv

libdv provides functions for programs interacting with the Quasar DV CODEC.
