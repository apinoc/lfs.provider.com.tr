ALSA Firmware-1.0.7

Introduction to ALSA Firmware

The ALSA Firmware package contains firmware for certain sound cards.

Package information

  • Download (HTTP): http://gd.tuwien.ac.at/opsys/linux/alsa/firmware/alsa-firmware-1.0.7.tar.bz2

  • Download (FTP): ftp://ftp.alsa-project.org/pub/firmware/alsa-firmware-1.0.7.tar.bz2

  • Download MD5 sum: 91b29697c758406d3b969006556fa606

  • Download size: 1007 KB

  • Estimated disk space required: 5.8 MB

  • Estimated build time: 0.03 SBU

ALSA Firmware dependencies

Required

ALSA Tools-1.0.7

Installation of ALSA Firmware

The ALSA Firmware package is only needed by those with advanced requirements for their sound card.

Install ALSA Firmware by running the following commands: 

./configure --prefix=/usr &&
make &&
make install
