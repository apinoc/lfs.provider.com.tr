EEL-2.8.2

Introduction to EEL

The EEL package contains the Eazel Extensions Library. This is a collection of widgets and extensions to the GNOME platform.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/eel/2.8/eel-2.8.2.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/eel/2.8/eel-2.8.2.tar.bz2

  • Download MD5 sum: 4d0beb2f0ad4c304b49c06b08d0a03e5

  • Download size: 620 KB

  • Estimated disk space required: 12.5 MB

  • Estimated build time: 0.43 SBU

EEL dependencies

Required

libgnomeui-2.8.0 and GAIL-1.8.0

Installation of EEL

Install EEL by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` &&
make &&
make install

Contents

The EEL package contains libeel libraries.

Description

libeel libraries

libeel libraries are a collection of widgets developed by the Nautilus project.
