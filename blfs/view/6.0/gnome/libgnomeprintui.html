libgnomeprintui-2.8.0

Introduction to libgnomeprintui

The libgnomeprintui package contains the libgnomeprintui libraries.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/libgnomeprintui/2.8/libgnomeprintui-2.8.0.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/libgnomeprintui/2.8/libgnomeprintui-2.8.0.tar.bz2

  • Download MD5sum: 0854339b4c40f9526e7430d36f98a785

  • Download size: 580 KB

  • Estimated disk space required: 14.2 MB

  • Estimated build time: 0.28 SBU

libgnomeprintui dependencies

Required

libgnomeui-2.8.0, GNOME Icon Theme-2.8.0 and libgnomeprint-2.8.0.1

Optional

GTK-Doc-1.2

Installation of libgnomeprintui

Install libgnomeprintui by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --disable-gtk-doc &&
make &&
make install

Command explanations

--disable-gtk-doc: This switch prevents rebuilding the documentation during the make command.
