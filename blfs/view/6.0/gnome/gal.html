GAL-2.2.3

Introduction to GAL

The GAL package contains library functions that came from Evolution and Gnumeric. GAL is short for GNOME Application Libs.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gal/2.2/gal-2.2.3.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gal/2.2/gal-2.2.3.tar.bz2

  • Download MD5 sum: a50985fd1b70dc05cfc921455c39e660

  • Download size: 1.2 KB

  • Estimated disk space required: 59 MB

  • Estimated build time: 1.24 SBU

GAL dependencies

Required

libgnomeprintui-2.8.0

Optional

GTK-Doc-1.2

Installation of GAL

Install GAL by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` &&
make &&
make install

Contents

The GAL package contains library routines pulled from the Evolution and Gnumeric projects and repackaged as a generic library.
