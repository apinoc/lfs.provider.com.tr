GDK Pixel Buffer-0.22.0

Introduction to GDK Pixel Buffer

The GDK Pixel Buffer package is the GTK+ pixel buffer library.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gdk-pixbuf/0.22/gdk-pixbuf-0.22.0.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gdk-pixbuf/0.22/gdk-pixbuf-0.22.0.tar.bz2

  • Download MD5 sum: 05fcb68ceaa338614ab650c775efc2f2

  • Download size: 396 KB

  • Estimated disk space required: 8.9 MB

  • Estimated build time: 0.32 SBU

GDK Pixel Buffer dependencies

Required

GTK+-1.2.10

Optional

GNOME Libraries-1.4.2, GTK-Doc-1.2 and DocBook-utils-0.6.14

Installation of GDK Pixel Buffer

The make command attempts to open an X display during the compile, so an X server must be running during this process.

Install GDK Pixel Buffer by running the following commands: 

./configure --prefix=/opt/gnome --disable-gtk-doc &&
make &&
make install

Contents

The GDK Pixel Buffer package contains libgdk_pixbuf and libgnomecanvaspixbuf libraries, along with many gdk-pixbuf loader modules.

Description

libgdk_pixbuf libraries

libdgk_pixbuf libraries contain the GTK+ pixel buffer libraries for the GIMP Toolkit.
