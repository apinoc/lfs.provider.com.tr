gtksourceview-1.1.1

Introduction to gtksourceview

The gtksourceview package contains libgtksourceview libraries. This is useful for extending the GTK text functions to include syntax highlighting.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gtksourceview/1.1/gtksourceview-1.1.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gtksourceview/1.1/gtksourceview-1.1.1.tar.bz2

  • Download MD5 sum: 2e59c8748594181d4bf452320c8c3b5c

  • Download size: 834 KB

  • Estimated disk space required: 11.7 MB

  • Estimated build time: 0.24 SBU

gtksourceview dependencies

Required

libgnomeprintui-2.8.0

Optional

GTK-Doc-1.2

Installation of gtksourceview

Install gtksourceview by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` &&
make &&
make install

Contents

The gtksourceview package contains libgtksourceview libraries.

Description

libgtksourceview libraries

libgtksourceview libraries contain function extensions for the GtkTextView widget.
