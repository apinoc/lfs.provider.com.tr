GNOME Additional Packages

These packages are modular and add desktop applications and assorted utilites to the GNOME environment. Feel free to install them on an as needed or as desired basis.

libgnomecups-0.1.13

Introduction to libgnomecups

The libgnomecups package contains a library used to wrap the CUPS API in a GLib fashion, so CUPS code can be cleanly integrated with GLib code.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/libgnomecups/0.1/libgnomecups-0.1.13.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/libgnomecups/0.1/libgnomecups-0.1.13.tar.bz2

  • Estimated build time: 0.09 SBU

libgnomecups dependencies

Required

GLib-2.6.3 and intltool-0.32.1

Installation of libgnomecups

Install libgnomecups by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` &&
make

Now, as the root user: 

make install
