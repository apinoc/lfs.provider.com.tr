GNOME Icon Theme-2.8.0

Introduction to GNOME Icon Theme

The GNOME Icon Theme package contains an assortment of scalable and non-scalable icons of different sizes and themes.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gnome-icon-theme/2.8/gnome-icon-theme-2.8.0.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gnome-icon-theme/2.8/gnome-icon-theme-2.8.0.tar.bz2

  • Download MD5 sum: 7be016337c44c024fb22f7b94b689d7b

  • Download size: 2.8 MB

  • Estimated disk space required: 27 MB

  • Estimated build time: 0.15 SBU

GNOME Icon Theme dependencies

Required

hicolor-icon-theme-0.5 and Perl modules: XML-Parser

Installation of GNOME Icon Theme

Install GNOME Icon Theme by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

The GNOME Icon Theme package contains icons.
