GNOME Virtual File System-1.0.5

Introduction to GNOME Virtual File System

The GNOME Virtual File System package contains file system libraries.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gnome-vfs/1.0/gnome-vfs-1.0.5.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gnome-vfs/1.0/gnome-vfs-1.0.5.tar.bz2

  • Download MD5 sum: e2a17a6b178f54c43968241258f3e729

  • Download size: 768 KB

  • Estimated disk space required: 18 MB

  • Estimated build time: 0.94 SBU

Additional downloads

  • Required Patch (if compiled using GCC-3.4.x): http://www.linuxfromscratch.org/blfs/downloads/6.0/gnome-vfs-1.0.5-gcc34-1.patch

GNOME Virtual File System dependencies

Required

GNOME MIME Data-2.4.2 and GNOME Libraries-1.4.2

Optional

pkgconfig-0.15.0, OAF-0.6.10, GConf-1.0.9, Bonobo-1.0.22, OpenSSL-0.9.7e, CDParanoia-III-9.8, GTK-Doc-1.2 and DocBook-utils-0.6.14

Installation of GNOME Virtual File System

Install GNOME Virtual File System by running the following commands: 

patch -Np1 -i ../gnome-vfs-1.0.5-gcc34-1.patch &&
./configure --prefix=/opt/gnome --disable-gtk-doc &&
make &&
make install

Contents

The GNOME Virtual File System package contains libgnomevfs libraries.
