libxklavier-1.11

Introduction to libxklavier

The libxklavier package contains a utility library for X keyboard.

Package information

  • Download (HTTP): http://prdownloads.sourceforge.net/gswitchit/libxklavier-1.11.tar.gz

  • Download (FTP):

  • Download MD5 sum: 1a4c6b95bc74bccaf5a7e394f928a54f

  • Download size: 425 KB

  • Estimated disk space required: 6.0 MB

  • Estimated build time: 0.11 SBU

libxklavier dependencies

Required

X (XFree86-4.4.0 or X.org-6.8.2), pkgconfig-0.15.0 and libxml2-2.6.17

Optional

Doxygen-1.4.1

Installation of libxklavier

Install libxklavier by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

The libxklavier package contains libxklavier libraries.
