gnome-session-3.20.2

Introduction to GNOME Session

The GNOME Session package contains the GNOME session manager.

This package is known to build and work properly using an LFS-7.10 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gnome-session/3.20/gnome-session-3.20.2.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gnome-session/3.20/gnome-session-3.20.2.tar.xz

  • Download MD5 sum: f2f71b012bae9f3a17b54b6384bc345a

  • Download size: 768 KB

  • Estimated disk space required: 18 MB

  • Estimated build time: 0.2 SBU

GNOME Session Dependencies

Required

dbus-glib-0.106, gnome-desktop-3.20.2, JSON-GLib-1.2.2, Mesa-12.0.1, and UPower-0.99.4

Optional

GConf-3.2.6 and libxslt-1.1.29 with docbook-xml-4.5 and docbook-xsl-1.79.1 (to build the documentation)

Runtime Dependencies

Systemd-231

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gnome-session

Installation of GNOME Session

Install GNOME Session by running the following commands: 

./configure --prefix=/usr --docdir=/usr/share/doc/gnome-session-3.20.2 &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Command Explanations

--enable-docbook-docs: This switch is used to enable building the documentation.

Contents

Installed Programs: gnome-session, gnome-session-inhibit, and gnome-session-quit
Installed Libraries: None
Installed Directories: /usr/share/doc/gnome-session-3.20.2, /usr/share/gnome-session, and /usr/share/wayland-sessions

Short Descriptions

gnome-session

is used to start up the GNOME Desktop environment.

gnome-session-inhibit

is used to inhibit certain GNOME Session functionality while executing the given command.

gnome-session-quit

is used to end the GNOME Session.
