MuPDF-1.12.0

Introduction to MuPDF

MuPDF is a lightweight PDF and XPS viewer.

This package is known to build and work properly using an LFS-8.2 platform.

Package Information

  • Download (HTTP): http://www.mupdf.com/downloads/archive/mupdf-1.12.0-source.tar.gz

  • Download MD5 sum: d30738f0487d2e40c921071ba74c479b

  • Download size: 49 MB

  • Estimated disk space required: 356 MB

  • Estimated build time: 0.5 SBU

Additional Downloads

  • Recommended patch: http://www.linuxfromscratch.org/patches/blfs/8.2/mupdf-1.12.0-consolidated_fixes-1.patch

  • Recommended patch (security fix): http://www.linuxfromscratch.org/patches/blfs/8.2/mupdf-1.12.0-security_fix-1.patch

Installation of MuPDF

Install MuPDF by running the following commands: 

rm -rf thirdparty/curl     &&
rm -rf thirdparty/freeglut &&
rm -rf thirdparty/freetype &&
rm -rf thirdparty/harfbuzz &&
rm -rf thirdparty/jpeg     &&
rm -rf thirdparty/lcms2    &&
rm -rf thirdparty/libjpeg  &&
rm -rf thirdparty/openjpeg &&
rm -rf thirdparty/zlib     &&

patch -Np1 -i ../mupdf-1.12.0-consolidated_fixes-1.patch &&
patch -Np1 -i ../mupdf-1.12.0-security_fix-1.patch &&

make build=release

This package does not come with a test suite.

Now, as the root user: 

make prefix=/usr                      \
     build=release                    \
     docdir=/usr/share/doc/mupdf-1.12.0 \
     install                          &&

ln -sfv mupdf-x11-curl /usr/bin/mupdf

Command Explanations

ln -sfv mupdf-x11-curl /usr/bin/mupdf : Three mupdf executables are installed: mupdf-gl, mupdf-x11, and mupdf-x11-curl. This symbolic link allows to choose which one is used when running mupdf.

Contents

Installed Program: mjsgen, mujstest, mupdf(symlink), mupdf-gl, mupdf-x11, mupdf-x11-curl, muraster, and mutool
Installed Libraries: libmupdf.so, libmupdfthird.so, and libmuthreads.so
Installed Directories: /usr/include/mupdf, /usr/share/doc/mupdf-1.12.0

Short Descriptions

mjsgen

is a test program for generating mjs files from pdf ones.

mujstest

is a test program for mupdf + javascript

mupdf

is a program for viewing PDF, XPS, EPUB, and CBZ documents, and various image formats such as PNG, JPEG, GIFF, and TIFF.

mupdf-gl

same as mupdf, using an opengl renderer.

mupdf-x11

same as mupdf, using an X Window renderer.

mupdf-x11-curl

same as mupdf, using an X Window renderer, with the capability of rendering web pages, using an “http://...” url.

muraster

transforms files in one of the formats displayable by mupdf into a graphic file.

mutool

deals with document files in various manners.

libmupdf.so

contains the mupdf API functions.

libmupdfthird.so

contains the third party libraries compiled for mudpf.
