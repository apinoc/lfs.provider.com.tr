Libkscreen-5.12.1 for lxqt

Introduction to libkscreen

The Libkscreen package contains the KDE Screen Management library.

This package is known to build and work properly using an LFS-8.2 platform.

Note

This package is extracted from the KDE Plasma set of packages. If Plasma-5.12.1 is built, do NOT also build this package as presented here.

Package Information

  • Download (HTTP): http://download.kde.org/stable/plasma/5.12.1/libkscreen-5.12.1.tar.xz

  • Download MD5 sum: cee80062a266ff50b120c3fccc2da4af

  • Download size: 96 KB

  • Estimated disk space required: 9.3 MB

  • Estimated build time: 0.3 SBU (using parallelism=4)

Libkscreen Dependencies

Required

kwayland-5.43.0 for lxqt

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/kwayland

Installation of libkscreen

Install libkscreen by running the following commands: 

mkdir -v build &&
cd       build &&

cmake -DCMAKE_INSTALL_PREFIX=$LXQT_PREFIX \
      -DCMAKE_BUILD_TYPE=Release          \
      -DCMAKE_INSTALL_LIBDIR=lib          \
      -DBUILD_TESTING=OFF                 \
      -Wno-dev ..                         &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Library: libKF5Screen.so
Installed Directories: $LXQT_PREFIX/lib/cmake/KF5ScreenConfig, $LXQT_PREFIX/lib/libexec/kf5, $LXQT_PREFIX/include/KF5/KScreen, and $LXQT_PREFIX/lib/plugins/kf5/kscreen
