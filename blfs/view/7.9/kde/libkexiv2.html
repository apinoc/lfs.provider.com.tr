libkexiv2-15.08.2

Introduction to libkexiv2

Libkexiv2 is a KDE wrapper around the Exiv2 library for manipulating image metadata.

This package is known to build and work properly using an LFS-7.9 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/applications/15.08.2/src/libkexiv2-15.08.2.tar.xz

  • Download MD5 sum: 2c30e2ac942e02a9d07e08f42958598e

  • Download size: 136 KB

  • Estimated disk space required: 5.3 MB

  • Estimated build time: 0.3 SBU

libkexiv2 Dependencies

Required

kdelibs-4.14.16 and Exiv2-0.25

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/libkexiv2

Installation of libkexiv2

Install libkexiv2 by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KDE_PREFIX   \
      -DCMAKE_BUILD_TYPE=Release           \
      -DCMAKE_MINIMUM_REQUIRED_VERSION=3.0 \
      -Wno-dev .. &&
make

Now, as the root user: 

make install

Contents

Installed Programs: none
Installed Library: libkexiv2.so
Installed Directories: $KDE_PREFIX/include/libkexiv2 and $KDE_PREFIX/share/apps/libkexiv2
