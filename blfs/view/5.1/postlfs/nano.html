nano-1.2.3

Introduction to nano

The nano package contains a small, simple text editor which aims to replace Pico, the default editor in the Pine package.

Package information

  • Download (HTTP): http://www.nano-editor.org/dist/v1.2/nano-1.2.3.tar.gz

  • Download (FTP): ftp://ftp.uni-koeln.de/editor/nano-1.2.3.tar.gz

  • Download size: 942 KB

  • Estimated Disk space required: 4.6 MB

  • Estimated build time: 0.08 SBU

nano dependencies

Optional

slang-1.4.9

Installation of nano

Install nano by running the following commands: 

./configure --prefix=/usr --sysconfdir=/etc \
    --enable-color --enable-multibuffer --enable-nanorc &&
make &&
make install &&
mkdir -p /usr/share/doc/nano/examples &&
cp nanorc.sample /usr/share/doc/nano/examples

Configuring nano

Config files

/etc/nanorc, ~/.nanorc

Example Configuration 

set autoindent
set const
set fill 72
set historylog
set multibuffer
set nohelp
set regexp
set smooth
set suspend

Another example is in the /usr/share/doc/nano/examples directory in the nanorc.sample file. It includes color configurations and has some documentation included in the comments.
