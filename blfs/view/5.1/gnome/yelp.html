Yelp-2.6.1

Introduction to Yelp

The Yelp package contains the help browser. This is useful for viewing help files.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/yelp/2.6/yelp-2.6.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/yelp/2.6/yelp-2.6.1.tar.bz2

  • Download size: 504 KB

  • Estimated Disk space required: 12 MB

  • Estimated build time: 0.16 SBU

Yelp dependencies

Required

libgnomeui-2.6.1.1 and libgtkhtml-2.6.1

Installation of Yelp

Install Yelp by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --libexecdir=`pkg-config --variable=prefix ORBit-2.0`/sbin &&
make &&
make install

Command explanations

--libexecdir=`pkg-config --variable=prefix ORBit-2.0`/sbin: This switch puts libexec files in $GNOME_PREFIX/sbin instead of $GNOME_PREFIX/libexec.
