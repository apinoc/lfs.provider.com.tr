GNOME Themes-2.6.1

Introduction to GNOME Themes

The GNOME Themes package contains several more theme sets.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gnome-themes/2.6/gnome-themes-2.6.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gnome-themes/2.6/gnome-themes-2.6.1.tar.bz2

  • Download size: 2.7 MB

  • Estimated Disk space required: 30.4 MB

  • Estimated build time: 0.37 SBU

GNOME Themes dependencies

Required

intltool-0.30 and GTK Engines-2.2.0

Installation of GNOME Themes

Install GNOME Themes by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

Contents

The GNOME Themes package contains themes.
