gst-plugins-0.8.1

Introduction to gst-plugins

The gst-plugins package contains CODEC interfaces.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gst-plugins/0.8/gst-plugins-0.8.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gst-plugins/0.8/gst-plugins-0.8.1.tar.bz2

  • Download size: 1.8 MB

  • Estimated Disk space required: 110 MB

  • Estimated build time: 2.16 SBU

gst-plugins dependencies

Required

GStreamer-0.8.1

Optional

NASM-0.98.38, liba52-0.7.4, AAlib-1.4rc5, ALSA-1.0.4, aRts-1.2.2, Audio File-0.2.6, CDParanoia-III-9.8, libdvdread-0.9.4, EsounD-0.2.34, FLAC-1.1.0, LAME-3.95.1, libdv-0.101, libFAME-0.9.1, libmad-0.15.1b, libmikmod-3.1.10, NAS-1.6, SDL-1.2.7, Speex-1.0.3, libogg-1.1, libvorbis-1.0.1, XviD-1.0.0-rc4, liboil, EM8300, V4L2, DivX4Linux, libdvdnav, FAAC, FAAD1 or FAAD2, GSM, Hermes, JACK, LADSPA, libcaca, libmpeg2, MJPEG Tools, libmusicbrainz, libraw1394, libshout, libsidplay, libsndfile, swfdec and Theora

Installation of gst-plugins

Install gst-plugins by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --sysconfdir=/etc/gnome &&
make &&
make install &&
gst-register

Contents

The gst-plugins package contains libgst* libraries.
