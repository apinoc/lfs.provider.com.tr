GTK Engines-2.2.0

Introduction to GTK Engines

The GTK Engines package contains three themes for GTK2.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gtk-engines/2.2/gtk-engines-2.2.0.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gtk-engines/2.2/gtk-engines-2.2.0.tar.bz2

  • Download size: 400 KB

  • Estimated Disk space required: 5.1 MB

  • Estimated build time: 0.10 SBU

GTK Engines dependencies

Required

GTK+-2.4.1

Installation of GTK Engines

Install GTK Engines by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

Contents

The GTK Engines package contains engines libraries and theme files.

Description

engines libraries

engines libraries are manager systems for specific themes.

theme files

Theme files included are Pixmap, Metal and Redmond95.
