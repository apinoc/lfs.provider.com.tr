gedit-2.6.1

Introduction to gedit

The gedit package contains a lightweight UTF-8 text editor for the GNOME desktop.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gedit/2.6/gedit-2.6.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gedit/2.6/gedit-2.6.1.tar.bz2

  • Download size: 2.1 MB

  • Estimated Disk space required: 59 MB

  • Estimated build time: 0.59 SBU

gedit dependencies

Required

EEL-2.6.1 and gtksourceview-1.0.1

Installation of gedit

Install gedit by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --localstatedir=/var/lib --sysconfdir=/etc/gnome &&
make &&
make install

Command explanations

--localstatedir=/var/lib: This switch puts ScrollKeeper files in /var/lib/scrollkeeper instead of $GNOME_PREFIX/var/scrollkeeper.

--sysconfdir=/etc/gnome: This switch puts configuration files in /etc/gnome instead of $GNOME_PREFIX/etc.
