GNOME Icon Theme-1.2.1

Introduction to GNOME Icon Theme

The GNOME Icon Theme package contains an assortment of scalable and non-scalable icons of different sizes and themes.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gnome-icon-theme/1.2/gnome-icon-theme-1.2.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gnome-icon-theme/1.2/gnome-icon-theme-1.2.1.tar.bz2

  • Download size: 2.7 MB

  • Estimated Disk space required: 13.7 MB

  • Estimated build time: 0.01 SBU

GNOME Icon Theme dependencies

Required

hicolor-icon-theme-0.5

Installation of GNOME Icon Theme

Install GNOME Icon Theme by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

Contents

The GNOME Icon Theme package contains icons.
