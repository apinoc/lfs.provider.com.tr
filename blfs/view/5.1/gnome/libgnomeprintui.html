libgnomeprintui-2.6.1

Introduction to libgnomeprintui

The libgnomeprintui package contains the libgnomeprintui libraries.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/libgnomeprintui/2.6/libgnomeprintui-2.6.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/libgnomeprintui/2.6/libgnomeprintui-2.6.1.tar.bz2

  • Download size: 550 KB

  • Estimated Disk space required: 31 MB

  • Estimated build time: 0.29 SBU

libgnomeprintui dependencies

Required

libgnomeui-2.6.1.1, GNOME Icon Theme-1.2.1 and libgnomeprint-2.6.1

Optional

GTK-Doc-1.2

Installation of libgnomeprintui

Install libgnomeprintui by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --disable-gtk-doc &&
make &&
make install

Command explanations

--disable-gtk-doc: This switch prevents rebuilding the documentation during the make command.

Contents

libgnomeprintui libraries

libgnomeprintui libraries are the GUI portion of the implementation of the GNOME Printing Architecture.
