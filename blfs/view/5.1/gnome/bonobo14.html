Bonobo-1.0.22

Introduction to Bonobo

The Bonobo package contains a set of language and system independent CORBA interfaces for creating reusable components, controls and compound documents.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/bonobo/1.0/bonobo-1.0.22.tar.bz2

  • Download (FTP): http://ftp.gnome.org/pub/GNOME/sources/bonobo/1.0/bonobo-1.0.22.tar.bz2

  • Download size: 1.2 MB

  • Estimated Disk space required: 110.2 MB

  • Estimated build time: 2.18 SBU

Bonobo dependencies

Required

GNOME Print-0.37

Installation of Bonobo

Install Bonobo by running the following commands: 

./configure --prefix=/opt/gnome &&
make &&
make install

Command explanations

--prefix=/opt/gnome: Install Bonobo in the GNOME 1.4 directory structure.

Contents

The Bonobo package contains GNOME 1.4 compound document and component libraries.
