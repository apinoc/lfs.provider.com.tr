EEL-2.6.1

Introduction to EEL

The EEL package contains the Eazel Extensions Library. This is a collection of widgets and extensions to the GNOME platform.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/eel/2.6/eel-2.6.1.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/eel/2.6/eel-2.6.1.tar.bz2

  • Download size: 542 KB

  • Estimated Disk space required: 22 MB

  • Estimated build time: 0.76 SBU

EEL dependencies

Required

libgnomeui-2.6.1.1 and GAIL-1.6.3

Installation of EEL

Install EEL by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` &&
make &&
make install

Contents

The EEL package contains libeel libraries.

Description

libeel libraries

libeel libraries are a collection of widgets developed by the Nautilus project.
