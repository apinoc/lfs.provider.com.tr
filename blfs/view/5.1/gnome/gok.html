GOK-0.10.2

Introduction to GOK

GOK is a dynamic onscreen keyboard. It features Direct Selection, Dwell Selection, Automatic Scanning and Inverse Scanning access methods and includes word completion.

Package information

  • Download (HTTP): http://ftp.gnome.org/pub/GNOME/sources/gok/0.10/gok-0.10.2.tar.bz2

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gok/0.10/gok-0.10.2.tar.bz2

  • Download size: 1.1 MB

  • Estimated Disk space required: 53 MB

  • Estimated build time: 1.01 SBU

GOK dependencies

Required

libgnomeui-2.6.1.1, ScrollKeeper-0.3.14, libwnck-2.6.1, AT SPI-1.4.2 and EsounD-0.2.34

Optional

GTK-Doc-1.2

Installation of GOK

Install GOK by running the following commands: 

./configure --prefix=`pkg-config --variable=prefix ORBit-2.0` \
    --sysconfdir=/etc/gnome --localstatedir=/var/lib &&
make &&
make install

Command explanations

--sysconfdir=/etc/gnome: This switch puts configuration files in /etc/gnome instead of $GNOME_PREFIX/etc.

--localstatedir=/var/lib: This switch puts ScrollKeeper files in /var/lib/scrollkeeper instead of $GNOME_PREFIX/var/scrollkeeper.
