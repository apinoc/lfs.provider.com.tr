w3m-0.4.2

Introduction to w3m

w3m is primarily a pager but it can also be used as a text-mode WWW browser.

Package information

  • Download (HTTP): http://prdownloads.sourceforge.net/w3m/w3m-0.4.2.tar.gz

  • Download (FTP):

  • Download size: 646 KB

  • Estimated Disk space required: 11 MB

  • Estimated build time: 0.28 SBU

w3m dependencies

Required

GC

Optional

GPM-1.20.1, OpenSSL-0.9.7d, Imlib-1.9.14, Imlib2-1.1.0, GDK Pixel Buffer-0.22.0 and Compface-1.4

Installation of w3m

Install w3m by running the following commands: 

./configure --prefix=/usr --libexecdir=/usr/lib --sysconfdir=/etc &&
make &&
make install &&
cp doc/keymap.default /etc/w3m/keymap &&
cp doc/menu.default /etc/w3m/menu

Configuring w3m

Config files

/etc/w3m/*, ~/.w3m/*

Contents

The w3m package contains w3m and w3mman.

Description

w3m

w3m is a text based web browser and pager.

w3mman

w3mman is an interface to the on-line reference manuals in w3m.
