kdeaddons-3.2.1

Introduction to kdeaddons

kdeaddons contains additional plugins and scripts for some KDE applications. This package should be compiled last.

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.2.1/src/kdeaddons-3.2.1.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.2.1/src/kdeaddons-3.2.1.tar.bz2

  • Download size: 1.4 MB

  • Estimated Disk space required: 32 MB

  • Estimated build time: 4.62 SBU

kdeaddons dependencies

Required

kdebase-3.2.1

Optional

libxml2-2.6.9, Berkeley DB-4.2.52.2, XMMS-1.2.10 and SDL-1.2.7

Installation of kdeaddons

Install kdeaddons with: 

./configure --prefix=/opt/kde-3.2.1 --disable-debug \
    --disable-dependency-tracking &&
make &&
make install

Contents

The KDE Addons package provides konqueror and noatun plugins, kicker applets and newskicker scripts.
