kdeutils-3.2.1

Introduction to kdeutils

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.2.1/src/kdeutils-3.2.1.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.2.1/src/kdeutils-3.2.1.tar.bz2

  • Download size: 2.5 MB

  • Estimated Disk space required: 41 MB

  • Estimated build time: 3.97 SBU

kdeutils dependencies

Required

kdebase-3.2.1

Optional

libxml2-2.6.9 and tpctl

Installation of kdeutils

Install kdeutils with: 

./configure --prefix=/opt/kde-3.2.1 --disable-debug \
    --disable-dependency-tracking &&
make &&
make install

Contents

Major programs in kdeutils include ark, irkick, kcalc, kcharselect, kcmlirc, kdepasswd, kdf, kedit, kfloppy, kgpg, khexedit, kjots, ksim, kregexpeditor, ktimer and kwallet.

Description

ark

ark is an archiving tool.

irkick

irkick is the infrastructure for the KDE's Infrared Remote Control functionality; irkick is the server component of that infrastructure.

kcalc

kcalc is a scientific calculator.

kcharselect

kcharselect is a character selector applet.

kdepasswd

kdepasswd is a password managing utility.

kdf

kdf is a disk usage viewer.

kedit

kedit is a text editor.

kfloppy

kfloppy is a floppy formatter.

kgpg

kgpg is a simple graphical interface for GnuPG-1.2.4.

khexedit

khexedit is a binary editor.

kjots

kjots is a note taker.

kregexpeditor

kregexpeditor is an editor for editing regular expressions in a graphical style (in contrast to the ASCII syntax).

ktimer

ktimer is a task scheduler.
