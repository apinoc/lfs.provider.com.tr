kdetoys-3.2.1

Introduction to kdetoys

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.2.1/src/kdetoys-3.2.1.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.2.1/src/kdetoys-3.2.1.tar.bz2

  • Download size: 2.4 MB

  • Estimated Disk space required: 14 MB

  • Estimated build time: 0.96 SBU

kdetoys dependencies

Required

kdebase-3.2.1

Optional

libxml2-2.6.9, libtiff-3.6.1 and JasPer,

Installation of kdetoys

Install kdetoys with: 

./configure --prefix=/opt/kde-3.2.1 --disable-debug \
    --disable-dependency-tracking &&
make &&
make install

Contents

Major programs in kdetoys include amor, kmoon, kodo, kteatime, ktux, kweather and kworldclock.

Description

amor

amor - Amusing Misuse of Resources.

kmoon

kmoon is a Moon phase indicator.

kodo

kodo measures your desktop mileage.

kteatime

kteatime times your tea brewing.

ktux

ktux - small Tux crossing stars.
