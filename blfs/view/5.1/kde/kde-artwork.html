kdeartwork-3.2.1

Introduction to kdeartwork

kdeartwork contains additional themes, screensaver, sounds, wallpapers and widget styles for KDE.

Package information

  • Download (HTTP): http://mirrors.isc.org/pub/kde/stable/3.2.1/src/kdeartwork-3.2.1.tar.bz2

  • Download (FTP): ftp://ftp.kde.org/pub/kde/stable/3.2.1/src/kdeartwork-3.2.1.tar.bz2

  • Download size: 17 MB

  • Estimated Disk space required: 57 MB

  • Estimated build time: 1.94 SBU

kdeartwork dependencies

Required

kdebase-3.2.1

Optional

libxml2-2.6.9, libart_lgpl-2.3.16 and xscreensaver

Installation of kdeartwork

Install kdeartwork with: 

./configure --prefix=/opt/kde-3.2.1 --disable-debug \
    --disable-dependency-tracking &&
make &&
make install

Contents

Major programs in kdeartwork include kscreensaver.

Description

kscreensaver

kscreensaver is the KDE screensaver.
