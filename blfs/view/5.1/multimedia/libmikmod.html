libmikmod-3.1.10

Introduction to libmikmod

libmikmod is a sound library capable of playing audio samples as well as tracker modules. Supported module formats include MOD, S3M, XM, IT, MED, MTM and 669.

Package information

  • Download (HTTP): http://www.ibiblio.org/pub/Linux/apps/sound/libs/libmikmod-3.1.10.tar.gz

  • Download (FTP):

  • Download size: 520 KB

  • Estimated Disk space required: 11 MB

  • Estimated build time: 0.31 SBU

libmikmod dependencies

Optional

EsounD-0.2.34

Installation of libmikmod

Install libmikmod by running the following commands: 

./configure --prefix=/usr &&
make &&
make install

Contents

The libmikmod package contains libmikmod-config and the libmikmod library.

Description

libmikmod-config

libmikmod-config provides version information, compiler and linker flags to programs that utilize libmikmod.

libmikmod

libmikmod provides the functionality that is required to play various tracker module files.
