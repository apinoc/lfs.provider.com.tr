slib-2d6

Introduction to slib

The slib package is a Scheme library used with Guile.

Package information

  • Download (HTTP): http://www.swiss.ai.mit.edu/ftpdir/scm/OLD/slib2d6.tar.gz

  • Download (FTP):

  • Download size: 636 KB

  • Estimated Disk space required: 2.9 MB

  • Estimated build time: 0.00 SBU

slib dependencies

Required

Guile-1.6.4

Installation of slib

Installation of slib is merely a copy command: 

cp -R slib /usr/share/guile

Contents

The slib package contains a Scheme library.
