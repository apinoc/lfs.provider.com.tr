adwaita-icon-theme-3.14.1

Introduction to Adwaita Icon Theme

The Adwaita Icon Theme package contains an icon theme for Gtk+ 3 applications.

This package is known to build and work properly using an LFS-7.7 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/adwaita-icon-theme/3.14/adwaita-icon-theme-3.14.1.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/adwaita-icon-theme/3.14/adwaita-icon-theme-3.14.1.tar.xz

  • Download MD5 sum: d03913d2d1624a6749a5c032082368fb

  • Download size: 17 MB

  • Estimated disk space required: 104 MB

  • Estimated build time: 1.1 SBU

Adwaita Icon Theme Dependencies

Required

GTK+-2.24.26 or GTK+-3.14.8 with librsvg-2.40.7 or both

Optional

git-2.3.0, Inkscape-0.91 and Icon Tools

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/adwaita-icon-theme

Installation of Adwaita Icon Theme

Install Adwaita Icon Theme by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: None
Installed Library: None
Installed Directory: /usr/share/icons/Adwaita/
