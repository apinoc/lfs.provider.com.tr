Folks-0.11.4

Introduction to Folks

Folks is a library that aggregates people from multiple sources (e.g, Telepathy connection managers and eventually Evolution Data Server, Facebook, etc.) to create metacontacts.

This package is known to build and work properly using an LFS-8.3 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/folks/0.11/folks-0.11.4.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/folks/0.11/folks-0.11.4.tar.xz

  • Download MD5 sum: 08a0e9759d577c6ece43e2db4e0d039e

  • Download size: 1.9 MB

  • Estimated disk space required: 93 MB (add 51 MB for tests)

  • Estimated build time: 1.0 SBU (add 2.4 SBU for tests)

Folks Dependencies

Required

evolution-data-server-3.28.5, gobject-introspection-1.56.1, libgee-0.20.1, and telepathy-glib-0.24.1

Recommended

BlueZ-5.50 and Vala-0.40.8

Optional

Tracker-2.0.4 (Broken due to API change), libsocialweb, and Zeitgeist

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/folks

Installation of Folks

Install Folks by running the following commands: 

./configure --prefix=/usr --disable-fatal-warnings &&
make

To test the results, issue make -k check from a local X session started with dbus-launch.

Now, as the root user: 

make install

Command Explanations

--disable-fatal-warnings: This switch allows the package to build with latest versions of the dependencies.

--enable-tracker-backend: This switch enables building the Tracker backend. Use it if you have Tracker installed. Note that this functionality is currently not working properly.

--enable-docs: This switch enables documentation generation.

Contents

Installed Programs: folks-import and folks-inspect
Installed Libraries: libfolks-dummy.so, libfolks-eds.so, libfolks.so, and libfolks-telepathy.so
Installed Directories: /usr/include/folks and /usr/lib/folks

Short Descriptions

folks-import

is used to import meta-contact information to libfolks.

folks-inspect

is used to inspect meta-contact information in libfolks.

libfolks-eds.so

contains Evolution specific implementations of the libfolks classes.

libfolks.so

contains the Folks API functions.

libfolks-telepathy.so

contains Telepathy specific implementations of the libfolks classes.

libfolks-tracker.so

contains Tracker specific implementations of the libfolks classes.
