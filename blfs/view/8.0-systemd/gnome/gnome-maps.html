gnome-maps-3.22.2

Introduction to GNOME Maps

GNOME Maps is a map application for GNOME.

This package is known to build and work properly using an LFS-8.0 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gnome-maps/3.22/gnome-maps-3.22.2.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gnome-maps/3.22/gnome-maps-3.22.2.tar.xz

  • Download MD5 sum: 7a1e0f10b00019024893368e3e80401f

  • Download size: 1.0 MB

  • Estimated disk space required: 13 MB

  • Estimated build time: less than 0.1 SBU

GNOME Maps Dependencies

Required

clutter-gtk-1.8.2, Folks-0.11.3, GeoClue-2.4.5, geocode-glib-3.20.1, gfbgraph-0.2.3, Gjs-1.46.0, and libchamplain-0.12.14

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gnome-maps

Installation of GNOME Maps

Install GNOME Maps by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: gnome-maps
Installed Libraries: libgnome-maps.so
Installed Directories: /usr/lib/gnome-maps and /usr/share/gnome-maps
