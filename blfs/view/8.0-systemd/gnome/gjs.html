Gjs-1.46.0

Introduction to Gjs

Gjs is a set of Javascript bindings for GNOME.

This package is known to build and work properly using an LFS-8.0 platform.

Package Information

  • Download (HTTP): http://ftp.gnome.org/pub/gnome/sources/gjs/1.46/gjs-1.46.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/gnome/sources/gjs/1.46/gjs-1.46.0.tar.xz

  • Download MD5 sum: 25574b60f4e1173fee0bd4920231df3e

  • Download size: 480 KB

  • Estimated disk space required: 68 MB (with tests)

  • Estimated build time: 0.5 SBU (with tests)

Gjs Dependencies

Required

Cairo-1.14.8, gobject-introspection-1.50.0, and JS2-24.2.0

Recommended (required for GNOME)

GTK+-3.22.8

Optional

Systemtap

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gjs

Installation of Gjs

Install Gjs by running the following commands: 

./configure --prefix=/usr &&
make

To test the results, issue: make check.

Now, as the root user: 

make install

Contents

Installed Programs: gjs (symlink) and gjs-console
Installed Library: libgjs.so
Installed Directories: /usr/include/gjs-1.0 and /usr/lib{,exec}/gjs

Short Descriptions

libgjs.so

contains the GNOME JavaScript bindings.
