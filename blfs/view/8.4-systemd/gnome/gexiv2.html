gexiv2-0.11.0

Introduction to gexiv2

gexiv2 is a GObject-based wrapper around the Exiv2 library.

This package is known to build and work properly using an LFS-8.4 platform.

Package Information

  • Download (HTTP): https://download.gnome.org/sources/gexiv2/0.11/gexiv2-0.11.0.tar.xz

  • Download (FTP): ftp://ftp.gnome.org/pub/GNOME/sources/gexiv2/0.11/gexiv2-0.11.0.tar.xz

  • Download MD5 sum: 6b9d0d9901df847a49903f11f3d6f685

  • Download size: 364 KB

  • Estimated disk space required: 5.1 MB

  • Estimated build time: 0.1 SBU

gexiv2 Dependencies

Required

Exiv2-0.27.0

Recommended

Vala-0.42.5

Optional

GTK-Doc-1.29 (for documentation) and Python-2.7.15 (for the Python2 module)

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gexiv2

Installation of gexiv2

Install gexiv2 by running the following commands: 

mkdir build &&
cd    build &&

meson --prefix=/usr .. &&
ninja

This package does not come with a test suite.

As the root user: 

ninja install

Contents

Installed Programs: None
Installed Libraries: libgexiv2.so
Installed Directories: /usr/include/gexiv2
