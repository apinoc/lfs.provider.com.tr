Gwenview-18.12.2

Introduction to Gwenview

Gwenview is a fast and easy-to-use image viewer for KDE.

This package is known to build and work properly using an LFS-8.4 platform.

Package Information

  • Download (HTTP): http://download.kde.org/stable/applications/18.12.2/src/gwenview-18.12.2.tar.xz

  • Download MD5 sum: 88267a7ead8638eed4ff8ba12e27a8e8

  • Download size: 5.5 MB

  • Estimated disk space required: 33 MB

  • Estimated build time: 0.7 SBU (using parallelism=4)

Additional Downloads

  • Required patch for building against Exiv2-0.27.0: http://www.linuxfromscratch.org/patches/blfs/8.4/gwenview-18.12.2-exiv2_0.27-1.patch

Gwenview Dependencies

Required

Exiv2-0.27.0, KDE Frameworks-5.55.0, and Little CMS-2.9

Recommended

libkdcraw-18.12.2

Optional

KF5Kipi

User Notes: http://wiki.linuxfromscratch.org/blfs/wiki/gwenview

Installation of Gwenview

First, fix several files so that Exiv2-0.27.0 is properly found and used: 

patch -Np1 -i ../gwenview-18.12.2-exiv2_0.27-1.patch

Install Gwenview by running the following commands: 

mkdir build &&
cd    build &&

cmake -DCMAKE_INSTALL_PREFIX=$KF5_PREFIX \
      -DCMAKE_BUILD_TYPE=Release         \
      -DBUILD_TESTING=OFF                \
      -Wno-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
