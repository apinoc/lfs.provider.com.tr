Current Development

This is the BLFS Book in its current development state. The instructions are generally stable, but changes can happen that break the build temporarily.

Daily rendered snapshots may be downloaded. Or check out the book's XML from the Git repository and render it yourself:

git clone git://git.linuxfromscratch.org/blfs.git blfs-git

Current Systemd Development

This is the BLFS Systemd Book in its current development state. The instructions are generally stable, but changes can happen that break the build temporarily.

Daily rendered snapshots may be downloaded. Or check out the book's XML from the Git repository and render it yourself using the instruction above.

After the prerequisites listed in the INSTALL file are installed, use 'make REV=systemd', to render the book. Additional information is listed in the README file.

Packages for BLFS

BLFS has a list of packages which you can download. Please use the versions which are listed in the book; these are tested versions which are known to work with each other. In addition to the packages you will need to download some patches that are also listed in the book. There are several ways to download the packages.

Hunt and Peck

When a package is not available from the location listed in the book, some other places to try are:

  • A search for the full package name in google or your favorite search engine
  • Debian Package Search Page

BLFS HTTP/FTP Sites

If the above methods are not available to you, the packages are also available individually on the following ftp and http mirrors.

  • ftp://ftp.lfs-matrix.net/pub/blfs/ (California, USA, 200Mbps)
  • http://ftp.lfs-matrix.net/pub/blfs/ (California, USA, 200Mbps)
  • ftp://ftp.osuosl.org/pub/blfs/ (Oregon, USA, 310Mbps)
  • http://ftp.osuosl.org/pub/blfs/ (Oregon, USA, 310Mbps)
