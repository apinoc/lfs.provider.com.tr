Errata for the 6.5 Version of the LFS Book

Miscellaneous Errata

  • In section 6.18.2, Contents of Ncurses, the program tabs is erroneously listed. This will be a part of a future version of Ncurses, but it is not currently installed.

  • At the end of section 6.31.1, Installation of Inetutils, there are several additional programs that should be moved to their FHS compliant location. The fragment should read as follows.

    Move some programs to their FHS-compliant place:

    mv -v /usr/bin/{hostname,ping,ping6} /bin
mv -v /usr/bin/traceroute /sbin

  • There are additional fixes for Bash-4.0 available at bash-4.0-fixes-6.patch.

    Important:This patch cannot be added as a part of the normal LFS build sequence because it changes a file that requires programs that are built later in the normal build sequence. The best way to apply these patches is to rebuild Bash-4.0 at the end of Chapter 6.
