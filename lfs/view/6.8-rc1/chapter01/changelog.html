1.3. Changelog

This is version 6.8-rc1 of the Linux From Scratch book, dated February 16, 2011. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://www.linuxfromscratch.org/mirrors.html.
