1.3. Changelog

This is version 10.1-rc1 of the Linux From Scratch book, dated February 19th, 2021. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://www.linuxfromscratch.org/mirrors.html.
